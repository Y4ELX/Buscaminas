Buscaminas
Este es un simple juego de Buscaminas creado utilizando HTML, CSS y JavaScript. El objetivo es descubrir todas las celdas que no contienen minas sin detonar ninguna.

Requisitos previos
Para ejecutar este proyecto, necesitarás lo siguiente:

Un navegador web moderno (Chrome, Firefox, Edge, etc.)
Un editor de texto o IDE (opcional, si deseas modificar el código)
Instrucciones para ejecutar el juego
Descargar el proyecto:

Clona el repositorio o descarga el proyecto como un archivo ZIP y extráelo en tu computadora.
bash
Copiar código
git clone https://github.com/Y4ELX/Buscaminas.git
Navegar a la carpeta del proyecto:

Abre la carpeta del proyecto donde se encuentran los archivos index.html, styles.css, script.js, y las carpetas img y sound.
Abrir el juego en un navegador:

Haz doble clic en el archivo index.html para abrirlo en tu navegador web preferido.
O

Abre el archivo index.html desde tu editor de texto y utiliza la opción de "Vista previa en el navegador" para abrir el juego.
Instrucciones del juego:

Haz clic en las celdas para revelar su contenido.
Si haces clic en una celda con una mina, perderás el juego.
Haz clic derecho en las celdas para marcarlas con una banderita si crees que contienen una mina.
El juego termina cuando has revelado todas las celdas que no contienen minas.
Reiniciar el juego:

Si pierdes, el juego se reiniciará automáticamente después de que todas las minas se revelen.
Personalización
Si deseas modificar el juego, puedes cambiar los siguientes archivos:

styles.css: Cambia el diseño y los colores de la cuadrícula y las celdas.
script.js: Modifica la lógica del juego, como el número de minas o el tamaño de la cuadrícula.
img y sound: Cambia las imágenes de las banderas, bombas y sonidos del juego.
